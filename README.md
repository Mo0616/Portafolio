# Portafolio - Michael Ordonez

Diversos proyectos realizados tanto personalmente como en mi universidad.

Este repositorio agrupa proyectos de portafolio alineados con mi hoja de vida como desarrollador full-stack en formacion, con experiencia en Python, JavaScript/React, C#, SQL, analisis de datos y optimizacion con Excel.

## Proyectos

| Proyecto | Tecnologia | Enfoque |
| --- | --- | --- |
| `python-crud-tareas` | Python + PostgreSQL | CRUD de tareas por consola con conexion a base de datos. |
| `csharp-biometrico` | C# + MySQL | Prototipo academico de autenticacion biometrica por comparacion de patrones. |
| `react-app-api` | React + JavaScript | Juego interactivo tipo quiz con consumo de API y fallback local. |
| `python-analisis-datos` | Python + CSV/HTML | Analisis de datos, seleccion de atributos e indicadores visuales. |
| `excel-solver` | Excel Solver + Python | Modelo de optimizacion para asignacion de recursos y reduccion de costos. |

## Objetivo

Mostrar evidencia minima y ordenada de los proyectos mencionados en mi hoja de vida, usando repositorios pequenos, documentados y faciles de ejecutar o revisar.

## Notas

- Los datos incluidos son ficticios y estan creados solo para demostracion.
- Cada carpeta contiene su propio `README.md` con instrucciones de uso.
