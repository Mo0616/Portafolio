# Juego Interactivo Diana - React + API

Aplicacion web interactiva tipo quiz. Usa `useState`, `useEffect`, renderizado condicional y consumo de API publica para cargar preguntas. Si la API no responde, usa preguntas locales de respaldo.

## Tecnologias

- React
- JavaScript
- Vite
- API publica Open Trivia DB
- CSS responsivo

## Ejecutar

```powershell
npm install
npm run dev
```

## Funcionalidades

- Carga de preguntas desde una API.
- Fallback local para que la app siga funcionando sin conexion.
- Puntuacion por respuestas correctas.
- Estados visuales para respuesta correcta e incorrecta.
- Reinicio de partida.
